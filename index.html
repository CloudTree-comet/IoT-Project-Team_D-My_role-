I2C Bus Optimization

Resource Efficiency Through Sharing

Designed a shared I2C bus (SDA=21, SCL=22) connecting both RTC and LCD. This architecture maximizes GPIO pin efficiency by controlling multiple peripherals with just two lines.

Power Isolation

Noise Immunity Architecture

Implemented separate power rails: 3.3V for ESP32 logic and 5V for servo motors. This prevents high-current noise from interfering with sensitive MCU logic.

Load Cell Sensing

0.1g High Precision

Integrated HX711 24-bit ADC amplifier. The system captures micro-volt signals from the load cell, enabling professional-grade food portion control.

Interactive Design

Pull-up Optimized UI

A intuitive 4-button interface using internal pull-up resistors, reducing wiring complexity while maintaining full control over feeding modes.

The Modular Framework

Software architecture that empowers team collaboration.

What is the Modular Framework?

It divides the firmware into independent, self-contained modules. This approach eliminated merge conflicts and enabled parallel development between UI, logic, and API tasks.

01. Data Integrity

Unified state variables ensure all modules read/write from synchronized memory locations.

02. Parallel Dev

Clear function hooks (coreLogicLoop, handleApiLoop) allow team-wide simultaneous coding.

03. 100% Validation

HAL layer allows testing logic path in Wokwi simulator before physical deployment.

Firmware Implementation

esp32_firmware_v2.0.cpp
// ---- ESP32 Pin Definition & Mapping ----
#define HX711_DT_PIN  4
#define HX711_SCK_PIN 5
#define SERVO_PIN     18

// UI Navigation Buttons
#define BUTTON_DISPLAY 12
#define BUTTON_SETTING 13
#define BUTTON_UP      14
#define BUTTON_DOWN    15

// Shared Hardware Objects
RTC_DS1307 rtc;
LiquidCrystal_I2C lcd(0x27, 20, 4);
Servo feedServo;
// ---- Global Shared Data Interface ----
struct FeedingSlot {
  bool  active;    // Slot enable flag
  int   hour;      // 0-23
  int   minute;    // 0-59
  float weight;    // Target portion (g)
};

FeedingSlot slots[3];
bool feedingActive = false;
float currentWeight = 0.0f;
// ---- Hardware Abstraction Layer ----
float readWeight(bool active, bool open) {
#if SIM_FAKE_WEIGHT
  if (active && open) {
    simWeight += 10.0f; // Virtual increase
  }
  return simWeight;
#else
  return scale.get_units(5);
#endif
}
// ---- Mechanical Control API ----
void openFeeder() {
  feedServo.write(180);
  feederOpen = true;
}

void resetSystemState() {
  feedingActive = false;
  currentWeight = 0;
  closeFeeder();
}
void setup() {
  Wire.begin(21, 22); // I2C Shared Bus
  lcd.init();
  feedServo.attach(SERVO_PIN);
  scale.begin(DT, SCK);
  scale.tare(); // Calibration
}
